## üîë QUICK SETUP GUIDE

### 1. Groq API (2 minutes)
```bash
1. Go to: https://console.groq.com
2. Sign up (email only, no credit card)
3. Click "API Keys"
4. Create new key
5. Copy: gsk_...
6. Add to backend/.env: GROQ_API_KEY=gsk_...
```

### 2. Cloudflare R2 (10 minutes)
```bash
1. Go to: https://dash.cloudflare.com
2. Sign up (email only)
3. Go to R2 Object Storage
4. Create bucket: "advision-creatives"
5. Go to "Manage R2 API Tokens"
6. Create API token (read & write)
7. Copy: Access Key ID & Secret Access Key
8. Add to backend/.env:
   R2_ACCESS_KEY_ID=...
   R2_SECRET_ACCESS_KEY=...
   R2_BUCKET_NAME=advision-creatives
   R2_ENDPOINT_URL=https://[account-id].r2.cloudflarestorage.com
```

### 3. Supabase (5 minutes)
```bash
1. Go to: https://supabase.com
2. Sign up (GitHub/email)
3. Create new project
4. Copy connection string
5. Add to backend/.env:
   DATABASE_URL=postgresql://...
```
